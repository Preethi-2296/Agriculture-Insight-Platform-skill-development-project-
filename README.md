🌾Agriculture-Insight-Platform
This application is developed using Zoho Creator.

📌 Project Overview
The Agricultural Insight Platform is a cloud-based smart farming application developed using Zoho Creator.  

It is designed to help farmers and agricultural analysts monitor soil health, environmental conditions, and crop recommendations using structured data collection and dashboard analytics.

The application is deployed using Zoho Creator Portal and supports authenticated multi-user access.

🚀 Key Features

📊 Interactive Dashboard
The dashboard provides real-time agricultural insights including:
- Average pH Levels
- Average Organic Matter
- Average Soil pH
- Average Humidity
- Soil Health Trends (Declining / Stable / Improving)
- Wind Speed vs Precipitation Analysis
- Phosphorus vs Nitrogen Visualization

🌱 Farm Data Management
- Add new farm data
- Store soil and environmental metrics
- Cloud-based data storage


🤖 AI Queries Module
- Submit AI-based agricultural queries
- Smart advisory interface


🧪 Soil Testing Module
- Add and manage soil test records
- Monitor soil quality indicators


🌾 Crop Recommendation System
- Add crop recommendations
- Analyze suitability based on soil and environmental data


🌦 Weather Forecast Module
- Track environmental conditions
- Support agricultural planning


🛠 Technologies Used
- Zoho Creator (Low-Code Development Platform)
- Deluge Scripting (Workflow & Automation)
- Zoho Cloud Database
- Dashboard Analytics & Data Visualization
- GitHub (Documentation & Deployment)
